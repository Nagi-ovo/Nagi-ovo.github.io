指针指的到底是什么

Created2023-12-10|Updated2023-12-10
|Word Count:1.9k|Reading Time:6mins|Post Views:

概念概述

首先，必须要弄明白C语言中的“变量类型”是什么。

其实它就是变量名前面的内容，又叫关键字。

比如int *p，那变量类型就是int *，即一个指向int类型的地址的指针。

同理，char *s; 声明了一个类型为 char * 的指针变量 s，它可以存储一个 char 类型变量（通常是一个字符数组或字符串）的地址。

明白了这点，语法理解层面就会减少很多困难了。

指针

  • 类比：将指针想象成书中的书签。它不包含书的内容，但指向特定的页面。
  • 在C语言中：指针是一个变量，它存储另一个变量的内存地址。例如，int *p; 意味着 p 是指向 int 类型的指针。

数组

  • 类比：数组就像一排排的储物柜，每个储物柜都有唯一的编号。
  • 在C语言中：数组是在连续的内存位置上放置的相同类型元素的集合。例如，int arr[5]; 创建了一个可以存储5个整数的数组。

字符串

  • 类比：字符串就像一串项链，每颗珠子代表一个字符。项链以一个特殊的珠子（空字符）结束。
  • 在C语言中：字符串是以空字符（\0）结束的字符数组。例如，char str[] = "Hello";。

指针、数组和字符串的互联

使用指针与数组

  • 例子：如果 arr 是一个数组，那么 arr 和 &arr[0] 是等价的——都指向第一个元素。
  • 访问元素：arr[i] 等同于 *(arr + i)。使用指针运算，你可以在数组中进行导航。

将字符串视为指针

  • 例子：char *str = "Hello"; —— 这里，str 是一个指向字符串第一个字符的指针。
  • 访问字符：可以使用 str[i] 或 *(str + i) 来访问字符。

而函数传参的基本原理在 C 语言中与 printf 函数是相同的，因为 printf 本质上也是一个函数。当你传递参数给任何函数时，你需要确保传递的参数类型与函数期望的参数类型相匹配。以下是一些基本的规则：

  1. 传递值：
    如果函数期望一个值（比如一个整数或字符），你直接传递变量本身。

  2. 传递指针：
    如果函数期望一个指针（它可能想要修改传入的变量的值，或者参数是一个数组），你需要传递变量的地址。

  3. 传递指针内容：
    如果你有一个指向某个数据的指针，而函数需要指针所指向的数据值，你需要传递解引用后的指针。

  4. 传递函数中的指针：
    如果函数期望一个 void * 类型的参数，你可以传递任何类型的指针，并且通常不需要显式类型转换，因为 C 语言允许任何类型的指针自动转换为 void *。

在 C 语言中传递指针时不会复制指针所指向的数据，而是复制指针本身（即地址）。这意味着函数内部可以通过传入的指针修改原始数据。这种传递方式称为“传址调用”（Call by Reference）。而直接传递值（不是地址）称为“传值调用”（Call by Value），在这种情况下，函数内部的修改不会影响原始数据。
